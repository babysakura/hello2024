# 

### 概要

ユーザーはスポット一覧観覧、お気に入りが出来ます。
お気に入りしたスポットはお気に入りページで観覧可能です。
管理者はスポットを投稿、編集、削除が可能でき、ユーザーの管理も出来きます。

### 主な機能
- ログイン・ログアウト機能
- 都道府県一覧画面
- スポット一覧画面
- ユーザー一覧画面
- お気に入り画面
- スポット登録、編集、削除、検索機能

### 開発環境
    ```
    PHP 8.2.10
    MySQL 5.7.39
    Laravel 10.38.2
    ```
